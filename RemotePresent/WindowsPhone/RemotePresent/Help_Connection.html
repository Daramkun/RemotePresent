리시버

이 앱을 사용하기 위해서는 "Remote Pack Receiver"가 필요합니다. 리시버의 다운로드 페이지는 "http://proj.daram.pe.kr"입니다. Windows 용이며, .NET Framework 4.0이 컴퓨터에 설치되어 있어야 합니다.

네트워크 상태

기기(윈도우폰)과 컴퓨터를 같은 네트워크 상으로 맞춰주어야 합니다(대부분의 경우, 같은 네트워크는 Wi-Fi를 뜻합니다).

장치 브로드캐스트

장치 이름을 설정합니다. 그리고 "브로드캐스트 시작" 버튼을 클릭합니다.

리시버에서 연결

브로드캐스트를 시작했다면 리시버에서 브로드캐스팅한 장치를 볼 수 있습니다. 해당 장치를 선택한 후 더블 클릭 혹은 엔터 키를 누르거나 연결하기 버튼을 클릭하면 패스워드 질문 창이 나타납니다. 패스워드를 입력하고 OK 버튼을 누르면 패스워드가 맞는 경우에 한해 접속에 성공합니다.
